👔 Mentor Dashboard Guide

Learn how to navigate and use your mentor dashboard effectively

Dashboard Overview

Your mentor dashboard is the central hub for managing internship positions, reviewing applications, and supervising interns. It provides a comprehensive view of all your mentoring activities.

Quick Access: Navigate to your dashboard anytime by clicking "Dashboard" in the top navigation menu or from the mentor-specific links.

Dashboard Sections

📊 Quick Statistics

At the top of your dashboard, you'll see:

  • Number of active positions you've created
  • Total applications received across all positions
  • Number of current active interns
  • Pending applications requiring your review

📝 Recent Positions

View your most recently created positions with:

  • Position title and department
  • Application deadline and start date
  • Number of applications received
  • Current status (Active, Closed, Draft)
  • Quick action buttons (Edit, View Applications, Manage)

📋 Pending Applications

Applications waiting for your review:

  • Student name and academic information
  • Position applied for
  • Application submission date
  • Quick preview of cover letter
  • Accept/Reject action buttons

👥 Current Interns

Overview of your active interns:

  • Intern names and contact information
  • Internship position and duration
  • Start and end dates
  • Last progress report submission
  • Quick actions (View Profile, Progress Reports, Contact)

🔔 Activity Feed

Stay updated with:

  • New applications received
  • Progress reports submitted by interns
  • Application deadlines approaching
  • Internship start/end date reminders
  • System notifications and updates

Company Status & Profile

⚠️ Important: Company Approval Status

Your dashboard will show your company's approval status:

  • Pending: Awaiting government official approval
  • Approved: Can create and manage positions
  • Rejected: Need to address issues and reapply
  • Suspended: Temporarily restricted access

Note: Only approved companies can create active positions.

Dashboard Best Practices

  • Check Daily: Review new applications and intern updates regularly
  • Respond Promptly: Students expect timely responses to their applications
  • Keep Positions Updated: Regularly review and update position details
  • Monitor Deadlines: Set reminders for application deadlines and internship milestones
  • Engage with Interns: Provide regular feedback and guidance
  • Plan Ahead: Create positions well in advance of start dates
  • Use Filters: Leverage dashboard filters to focus on priority items

Dashboard Customization

Personalize Your Experience:

  • Set notification preferences for applications and deadlines
  • Choose which quick stats to display prominently
  • Organize positions by priority or deadline
  • Customize the activity feed to show relevant updates
