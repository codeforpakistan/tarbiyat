Registration Process

1

Access Registration Form

  • Navigate to your teacher dashboard
  • Click "Register Organization" or access via direct URL: /register-organization/
  • Select "Institute" as organization type
2

Complete Institute Information

  • Institute Name: Official registered name
  • Description: Brief overview of educational programs
  • Address: Physical campus address
  • Website: Institute website URL (optional)
  • Contact Email: Official institutional email
  • Phone: Main contact number
  • Email Domain: Institute email domain for verification
3

Upload Required Documents

  • Registration Certificate: Official institute registration document
  • Authorization Letter: Letter authorizing you to represent the institute
  • Ensure documents are clear and readable
  • Supported formats: PDF, DOC, DOCX, JPG, PNG
4

Submit for Review

  • Review all information for accuracy
  • Submit registration request
  • You'll receive confirmation of submission
  • Track request status via "Organization Requests" page

Troubleshooting

"Access denied" when trying to register

Ensure you have a complete teacher profile and the can_register_organization permission. Contact support if the issue persists.

Document upload fails

Check file size (must be under 10MB) and format (PDF, DOC, DOCX, JPG, PNG). Ensure files are not corrupted.

Request was rejected

Check the official feedback in your request details. Address the concerns and submit a new request with corrected information.

Can't edit institute after approval

Only the teacher who registered the institute (admin contact) can edit institute details. Other teachers need to request changes through the admin.

Students can't join my institute

Ensure your institute has domain verification enabled and students are using emails from the correct domain. Check that the institute status is approved.
