What is Tarbiyat?

Tarbiyat is a comprehensive internship management platform designed specifically for government agencies. It streamlines the entire internship process from position posting to completion tracking.

User Roles

🎓 Students

Browse positions, submit applications, track progress

👔 Mentors

Create positions, review applications, supervise interns

👨‍🏫 Teachers

Academic supervision, coordinate with mentors

👮 Officials

Organization approvals, oversight, compliance monitoring

Key Features

  • Position management and posting
  • Application tracking and review
  • Progress monitoring and reporting
  • Organization verification system
  • Real-time notifications
  • Multi-role dashboard system
